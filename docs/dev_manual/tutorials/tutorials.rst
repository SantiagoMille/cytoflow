*********
Tutorials
*********

These tutorials illustrate some of the capabilities of the `cytoflow` library.
They are generated from the example notebooks and data sets in the 
``cytoflow-$VERSION-examples-basic.zip`` file, available from the ``Releases`` tab
at `the project homepage <https://github.com/cytoflow/cytoflow/>`_. 
Feel free to fire up a `Jupyter notebook <https://jupyter.org/>`_ and follow along!
